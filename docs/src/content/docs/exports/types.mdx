All types are importable from `colorizr`:

```typescript
import type { HSL, RGB, LAB, LCH, ColorType, Analysis } from 'colorizr';
```

## Color Models

### HSL

```typescript
interface HSL {
  h: number;  // Hue: 0-360
  s: number;  // Saturation: 0-100
  l: number;  // Lightness: 0-100
  alpha?: number; // 0-1
}
```

### RGB

```typescript
interface RGB {
  r: number;  // Red: 0-255
  g: number;  // Green: 0-255
  b: number;  // Blue: 0-255
  alpha?: number; // 0-1
}
```

### LAB (OkLab)

```typescript
interface LAB {
  l: number;  // Lightness: 0-1
  a: number;  // Green-red axis: -0.4 to 0.4
  b: number;  // Blue-yellow axis: -0.4 to 0.4
  alpha?: number; // 0-1
}
```

### LCH (OkLCH)

```typescript
interface LCH {
  l: number;  // Lightness: 0-1
  c: number;  // Chroma: 0-0.4
  h: number;  // Hue: 0-360
  alpha?: number; // 0-1
}
```

## Generic Helpers

Conditional/generic types used by the converters and color accessors.

### ColorReturn

Maps a `ColorType` to its concrete model — the return type of format-aware functions.

```typescript
type ColorReturn<T extends ColorType> =
  T extends 'hex' ? HEX
  : T extends 'hsl' ? HSL
  : T extends 'oklab' ? LAB
  : T extends 'oklch' ? LCH
  : T extends 'rgb' ? RGB
  : never;
```

### ColorModelKeys

The channel keys (excluding `alpha`) for a given color model.

```typescript
type ColorModelKeys<TModel extends ColorModelKey> =
  TModel extends 'hsl' ? keyof Omit<HSL, 'alpha'>
  : TModel extends 'oklab' ? keyof Omit<LAB, 'alpha'>
  : TModel extends 'oklch' ? keyof Omit<LCH, 'alpha'>
  : TModel extends 'rgb' ? keyof Omit<RGB, 'alpha'>
  : never;
```

### ConverterParameters

Accepted input for a converter: either the model object or its `ColorTuple`.

```typescript
type ConverterParameters<TModel extends ColorModel> = TModel | ColorTuple;
```

## Analysis

Returned by `compare()`:

```typescript
interface Analysis {
  brightnessDifference: number;
  colorDifference: number;
  compliant: number;       // 0-2, levels of compliance passed
  contrast: number;        // WCAG contrast ratio
  largeAA: boolean;
  largeAAA: boolean;
  normalAA: boolean;
  normalAAA: boolean;
}
```

### MixOptions

```typescript
interface MixOptions {
  format?: ColorType;
  space?: ColorModelKey;
  hue?: HueMode;

type HueMode = 'shorter' | 'longer' | 'increasing' | 'decreasing';
```

### ScaleOptions

```typescript
interface ScaleOptions {
  steps?: number;          // 3-20, default 11
  variant?: ScaleVariant;
  saturation?: number;     // 0-100
  mode?: ScaleMode;        // default 'light'
  lightnessCurve?: number | ScaleRange;            // default 1.5
  chromaCurve?: number | ScaleChromaPeak | ScaleRange; // default 0
  hueShift?: number | ScaleRange;                  // -180 to 180, default 0
  minLightness?: number;   // 0-1, default 0.2
  maxLightness?: number;   // 0-1, default 0.97
  lock?: number;
  format?: ColorType;
}

type ScaleMode = 'light' | 'dark' | 'reversed';

type ScaleVariant = 'deep' | 'neutral' | 'pastel' | 'subtle' | 'vibrant';

interface ScaleChromaPeak {
  amount: number;          // 0-1
  peak?: number;           // 0-1, default 0.5
}

interface ScaleRange {
  high: number;            // high-key end (950)
  low: number;             // low-key end (50)
}
```
